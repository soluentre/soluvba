tediouscharts
------------------------------------------------------
更新日志：
v1.0.0：基本功能完成开发和测试
  1. 介绍：用于批量循环生成大量简单同类型Excel图表（如柱形图，折线图，饼状图，堆叠柱形图等），支持图表标题，表中文本框内容的修改和样式同步。使用需要具备一定基础的数据处理技能，把要批量生成图表的数据处理成特定的长格式结构。
  2. 使用说明：
    1. 表格说明：
      1. data - 处理后数据承载页以及最终图表展示页；
      2. arrangement - 图表展示页的批量图表排列参数，例如图与图的纵向以及横向间隔，单行排列多少个图表等等；
      3. info - 需要批量生成的图表的详细信息，包括每张图需要替换的文字内容，需要标注的格式样式等等； 
      4. sample - 批量生成图表基于的模板，注意图表模板名应该和info表格填写的type列参数保持一致。
    2. 使用流程：
      1. 处理目标数据，并把处理后数据存入data表；
      2. 修改arrangement，info以及sample表参数内容；
      3. 运行名为CreateChartsFromTable的VBA宏，并检查运行结果，必要时根据结果进行参数调整后再次运行。
注意事项：
1. 目前支持图表自带标题的文字替换，但不支持格式复制；
2. 目前每张图表的文本框修改仅支持2个，一般默认一个为上方用于替代标题的boxtitle文本框，一个为下方用于记录数据来源的note文本框。从原则上可以通过简单的代码升级支持更多文本框的修改，以及文本框的位置固定等；
3. 如果需要制作多个面板，可以在制作前复制一份所有表的备份，然后在处理完成后把项目表格重命名，然后把备份表格命名回：data，arrangement，info和sample；因为目前代码的功能仅支持读取这四个表名作为程序的运行参数。
